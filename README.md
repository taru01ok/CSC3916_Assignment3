# CSC3916 Assignment 3

## Description
A REST API built with Node.js, Express, and MongoDB that supports user authentication with JWT tokens and CRUD operations for movies.

## Installation
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with `DB`, `SECRET_KEY`, and `PORT`
4. Run `node server.js`

## Deployed API
https://csc3916-assignment3-12ic.onrender.com

## Environment Variables
- `DB` - MongoDB connection string
- `SECRET_KEY` - JWT secret key
- `PORT` - Server port
